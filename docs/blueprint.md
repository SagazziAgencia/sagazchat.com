## Style Guidelines:

- Primary color: Strong green (#2962FF) to inspire trust and confidence.
- Background color: Very light green (#E8F0FE), almost white, to ensure readability and a clean aesthetic.
- Accent color: A vivid green (#7F56D9) to call attention to key details like calls to action. Green suggests the app's blending of business and interpersonal communications.
- Body text: 'Inter' sans-serif for clarity and readability.
- Headline font: 'Space Grotesk' sans-serif for headlines, complementing the body.
- Modern, flat-style icons to represent features and benefits.
- Clean and structured layout with clear sections.
- Subtle animations and transitions to enhance user engagement.

### Pricing UI Components:
- **Toggle Switch**: Botões Só WhatsApp / WhatsApp + Instagram com cores distintas
- **Plan Cards**: Cards com glassmorphism, bordas sutis, hover effects
