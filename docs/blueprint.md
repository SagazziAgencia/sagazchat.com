# **App Name**: Sagazchat

## Core Features

- **Headline and Value Proposition:** Clear and concise headline communicating the value of Sagazchat
- **Feature Highlights:** Showcase key features with descriptions and visuals (screenshots, mockups, or icons)
- **Social Proof:** Display testimonials, reviews, or case studies from satisfied users
- **Call to Action:** Prominent 'Sign Up' or 'Learn More' buttons
- **Contact Form:** Simple form to capture leads and user inquiries
- **AI-Powered Chat Tool:** AI assistant on the landing page to address common questions

## Pricing Section

### Platform Toggle

- **Só WhatsApp:** Plans for WhatsApp automation only
- **WhatsApp + Instagram:** Plans with combined WhatsApp and Instagram automation

### Plans Structure

- **Basic:** Essential features with WhatsApp, Webhooks, 10 users, 2 Kanbans
- **PRO:** Recommended plan with enhanced limits, Calendar, API Integrations, Onboarding call
- **Basic + IA:** Basic plan + 1 AI Agent
- **PRO + IA:** PRO plan + 2 AI Agents
- **Instagram Add-on:** Adds automated DMs and comment automation

### Connections

- All plans support **1 to 10 connections**
- Progressive savings with more connections

## Style Guidelines

### Color Palette

- **Primary:** `#92D639` (Vibrant Green) - Trust and energy
- **Background:** `#050505` (Deep Black) - Modern and premium
- **Accent:** `#FF8A00` (Orange) - Attention and calls to action
- **Surface:** Dark slate tones for cards and elevated elements
- **Text:** White and slate shades for optimal readability on dark backgrounds

### Typography

- **Body Text:** 'Inter' sans-serif for clarity and readability
- **Headlines:** Modern sans-serif with tight letter-spacing
- **Sizes:** Minimum 16px base for mobile accessibility

### UI Components

- **Modern glassmorphism** for cards and surfaces
- **Spring-based animations** for interactions (stiffness: 350, damping: 25)
- **Hover effects:** Lift (translateY: -2px) with enhanced shadows
- **Click effects:** Subtle scale (0.98) compression
- **4px grid system** for all spacing
- **Responsive design:** Mobile-first approach with progressive enhancement
