## Padrões Aprovados

### Landing Page — Oneshot aprovado (2026-04-05)
O agente gerou a landing page completa num oneshot. Iarley elogiou o resultado.

**O que funcionou:**
- Usar o vault como fonte de verdade antes de criar (preços reais, módulos reais)
- Atender dois perfis na mesma página: quem quer assinar + quem quer setup
- Seção de setup ("Não quer configurar sozinho? A gente monta com você.") inserida organicamente no fluxo, sem sujar o posicionamento de produto
- Copy direto com ponto de vista — ex: "Cada conversa sem resposta é uma venda que você perde."
- Features grid com módulos reais e descrições funcionais (sem buzzwords)
- Pricing com preços reais, toggle WhatsApp/Instagram, destaque no plano mais popular

**Protocolo:** Elogios do Iarley = salvar o padrão aqui para reuso futuro.
